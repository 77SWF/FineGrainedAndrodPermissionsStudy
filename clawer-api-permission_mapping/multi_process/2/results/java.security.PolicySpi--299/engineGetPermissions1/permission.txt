<p>Return a PermissionCollection object containing the set of
 permissions granted to the specified CodeSource.

 </p><p> The default implementation of this method returns
 Policy.UNSUPPORTED_EMPTY_COLLECTION object.  This method can be
 overridden if the policy implementation can return a set of
 permissions granted to a CodeSource.</p><table class="responsive">

          If this operation is supported, the returned
          set of permissions must be a new mutable instance

          Policy.UNSUPPORTED_EMPTY_COLLECTION is returned.<p></p></td>
