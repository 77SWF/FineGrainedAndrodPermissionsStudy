<p>Set the provider for a given appWidgetId if the caller has a permission.

 If successful, the app widget provider will receive a <code dir="ltr" translate="no"><a href="/reference/android/appwidget/AppWidgetManager#ACTION_APPWIDGET_UPDATE">ACTION_APPWIDGET_UPDATE</a></code>
 broadcast.

 </p><p>
 <strong>Note:</strong> You need the <code dir="ltr" translate="no"><a href="/reference/android/Manifest.permission#BIND_APPWIDGET">Manifest.permission.BIND_APPWIDGET</a></code>
 permission or the user must have enabled binding widgets always for your component.
 Should be used by apps that host widgets. If this method returns false, call <code dir="ltr" translate="no"><a href="/reference/android/appwidget/AppWidgetManager#ACTION_APPWIDGET_BIND">ACTION_APPWIDGET_BIND</a></code> to request permission to bind.
