 used to direct traffic to the network (although accessing some networks may be subject to
 holding specific permissions). Callers will learn about the specific characteristics of the

 <code dir="ltr" translate="no"><a href="/reference/android/net/ConnectivityManager.NetworkCallback#onLinkPropertiesChanged(android.net.Network,%20android.net.LinkProperties)">NetworkCallback#onLinkPropertiesChanged(Network, LinkProperties)</a></code>. The methods of the
 provided <code dir="ltr" translate="no"><a href="/reference/android/net/ConnectivityManager.NetworkCallback">NetworkCallback</a></code> will only be invoked due to changes in the best network
 matching the request at any given time; therefore when a better network matching the request
 becomes available, the <code dir="ltr" translate="no"><a href="/reference/android/net/ConnectivityManager.NetworkCallback#onAvailable(android.net.Network)">NetworkCallback#onAvailable(Network)</a></code> method is called
 with the new network after which no further updates are given about the previously-best
 network, unless it becomes the best again at some later time. All callbacks are invoked
 in order on the same thread, which by default is a thread created by the framework running
