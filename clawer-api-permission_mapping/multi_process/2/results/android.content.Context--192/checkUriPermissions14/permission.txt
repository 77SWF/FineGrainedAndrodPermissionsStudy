<p>Determine whether a particular process and user ID has been granted
 permission to access a list of URIs.  This only checks for permissions
 that have been explicitly granted -- if the given process/uid has
 more general access to the URI's content provider then this check will
 always fail.

 <strong>Note:</strong> On SDK Version <code dir="ltr" translate="no"><a href="/reference/android/os/Build.VERSION_CODES#S">Build.VERSION_CODES.S</a></code>,
 calling this method from a secondary-user's context will incorrectly return
 <code dir="ltr" translate="no"><a href="/reference/android/content/pm/PackageManager#PERMISSION_DENIED">PackageManager#PERMISSION_DENIED</a></code> for all {code uris}.</p><table class="responsive">
