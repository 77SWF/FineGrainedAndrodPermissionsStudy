<p>
 This allows apps with Storage Access Framework permissions to convert
 between <code dir="ltr" translate="no"><a href="/reference/android/provider/MediaStore">MediaStore</a></code> and <code dir="ltr" translate="no"><a href="/reference/android/provider/DocumentsProvider">DocumentsProvider</a></code> Uris that refer
 to the same underlying item. Note that this method doesn't grant any new
 permissions; callers must already hold permissions obtained with
 <code dir="ltr" translate="no"><a href="/reference/android/content/Intent#ACTION_OPEN_DOCUMENT">Intent#ACTION_OPEN_DOCUMENT</a></code> or related APIs.</p>
