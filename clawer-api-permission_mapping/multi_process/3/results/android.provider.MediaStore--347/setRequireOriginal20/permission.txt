<p>
 If the original file contents cannot be provided, a
 <code dir="ltr" translate="no"><a href="/reference/java/lang/UnsupportedOperationException">UnsupportedOperationException</a></code> will be thrown when the returned
 <code dir="ltr" translate="no"><a href="/reference/android/net/Uri">Uri</a></code> is used, such as when the caller doesn't hold
 <code dir="ltr" translate="no"><a href="/reference/android/Manifest.permission#ACCESS_MEDIA_LOCATION">Manifest.permission.ACCESS_MEDIA_LOCATION</a></code>.</p>
