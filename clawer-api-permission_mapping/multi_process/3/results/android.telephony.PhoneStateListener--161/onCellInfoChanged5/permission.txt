<p>Callback invoked when a observed cell info has changed or new cells have been added
 or removed on the registered subscription.
 Note, the registration subId s from <code dir="ltr" translate="no"><a href="/reference/android/telephony/TelephonyManager">TelephonyManager</a></code> object which registers
 PhoneStateListener by <code dir="ltr" translate="no"><a href="/reference/android/telephony/TelephonyManager#listen(android.telephony.PhoneStateListener,%20int)">TelephonyManager#listen(PhoneStateListener, int)</a></code>.
 If this TelephonyManager object was created with
 <code dir="ltr" translate="no"><a href="/reference/android/telephony/TelephonyManager#createForSubscriptionId(int)">TelephonyManager#createForSubscriptionId(int)</a></code>, then the callback applies to the
 subId. Otherwise, this callback applies to
 <code dir="ltr" translate="no"><a href="/reference/android/telephony/SubscriptionManager#getDefaultSubscriptionId()">SubscriptionManager#getDefaultSubscriptionId()</a></code>.

 <br/>
 Requires <code dir="ltr" translate="no"><a href="/reference/android/Manifest.permission#READ_PHONE_STATE">Manifest.permission.READ_PHONE_STATE</a></code> and <code dir="ltr" translate="no"><a href="/reference/android/Manifest.permission#ACCESS_FINE_LOCATION">Manifest.permission.ACCESS_FINE_LOCATION</a></code></p>
