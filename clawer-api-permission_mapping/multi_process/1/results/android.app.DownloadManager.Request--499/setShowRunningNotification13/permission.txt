<p>Control whether a system notification is posted by the download manager while this
 download is running. If enabled, the download manager posts notifications about downloads
 through the system <code dir="ltr" translate="no"><a href="/reference/android/app/NotificationManager">NotificationManager</a></code>. By default, a notification is
 shown.

 If set to false, this requires the permission
 android.permission.DOWNLOAD_WITHOUT_NOTIFICATION.</p>
