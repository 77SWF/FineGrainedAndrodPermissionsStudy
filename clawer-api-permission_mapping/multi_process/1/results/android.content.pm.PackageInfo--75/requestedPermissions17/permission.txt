<p>Array of all <code dir="ltr" translate="no"><a href="/reference/android/R.styleable#AndroidManifestUsesPermission">&lt;uses-permission&gt;</a></code> tags included under &lt;manifest&gt;,
 or null if there were none.  This is only filled in if the flag
 <code dir="ltr" translate="no"><a href="/reference/android/content/pm/PackageManager#GET_PERMISSIONS">PackageManager#GET_PERMISSIONS</a></code> was set.  This list includes
 all permissions requested, even those that were not granted or known
 by the system at install time.</p>
