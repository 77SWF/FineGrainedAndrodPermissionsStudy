<p>The caller must have the <code dir="ltr" translate="no"><a href="/reference/android/Manifest.permission#INTERACT_ACROSS_PROFILES">Manifest.permission.INTERACT_ACROSS_PROFILES</a></code>,
 <code dir="ltr" translate="no">android.Manifest.permission#INTERACT_ACROSS_USERS</code>, or <code dir="ltr" translate="no">android.Manifest.permission#INTERACT_ACROSS_USERS_FULL</code> permission. Both the caller and
 target user profiles must be in the same profile group. The target user must be a valid user
 returned from <code dir="ltr" translate="no"><a href="/reference/android/content/pm/CrossProfileApps#getTargetUserProfiles()">getTargetUserProfiles()</a></code>.
