                <a href="/reference/java/lang/String">String[]</a> permissions, 
                int[] grantResults)</pre><p>Callback for the result from requesting permissions. This method
 is invoked for every call on <code dir="ltr" translate="no"><a href="/reference/android/app/Activity#requestPermissions(java.lang.String[],%20int)">requestPermissions(java.lang.String[], int)</a></code>.

 <strong>Note:</strong> It is possible that the permissions request interaction
