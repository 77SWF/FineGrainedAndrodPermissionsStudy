<pre class="api-signature no-pretty-print" dir="ltr" translate="no">public void adoptShellPermissionIdentity (<a href="/reference/java/lang/String">String...</a> permissions)</pre><p>Adopt the permission identity of the shell UID only for the provided permissions.
 This allows you to call APIs protected permissions which normal apps cannot hold
 but are granted to the shell UID. If you already adopted shell permissions by calling
 this method, or <code dir="ltr" translate="no"><a href="/reference/android/app/UiAutomation#adoptShellPermissionIdentity()">adoptShellPermissionIdentity()</a></code> a subsequent call will replace any
 previous adoption.
 </p><p>
 <strong>Note:<strong> This method behave differently from
 <code dir="ltr" translate="no"><a href="/reference/android/app/UiAutomation#adoptShellPermissionIdentity()">adoptShellPermissionIdentity()</a></code>. Only the listed permissions will use the shell
 identity and other permissions will still check against the original UID</strong></strong></p><strong><strong>
