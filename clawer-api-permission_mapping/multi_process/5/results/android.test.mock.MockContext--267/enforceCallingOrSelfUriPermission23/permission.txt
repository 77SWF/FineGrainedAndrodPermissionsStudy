<p>If the calling process of an IPC <em>or you</em> has not been
 granted permission to access a specific URI, throw <code dir="ltr" translate="no"><a href="/reference/java/lang/SecurityException">SecurityException</a></code>.  This is the same as <code dir="ltr" translate="no"><a href="/reference/android/content/Context#enforceCallingUriPermission(android.net.Uri,%20int,%20java.lang.String)">enforceCallingUriPermission(Uri, int, String)</a></code>, except it grants your own
 permissions if you are not currently processing an IPC.  Use
 with care!</p>
