<p>Registers a <code dir="ltr" translate="no"><a href="/reference/android/telephony/ims/RegistrationManager.RegistrationCallback">RegistrationManager.RegistrationCallback</a></code> with the system. When the
 callback is registered, it will initiate the callback c to be called with the current
 registration state.

 Requires Permission: <code dir="ltr" translate="no"><a href="/reference/android/Manifest.permission#READ_PRECISE_PHONE_STATE">READ_PRECISE_PHONE_STATE</a></code> or that the calling app has carrier privileges
 (see <code dir="ltr" translate="no"><a href="/reference/android/telephony/TelephonyManager#hasCarrierPrivileges()">TelephonyManager.hasCarrierPrivileges()</a></code>).
