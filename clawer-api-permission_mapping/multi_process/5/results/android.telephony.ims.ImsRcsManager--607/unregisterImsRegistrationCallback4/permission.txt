<p>Removes an existing <code dir="ltr" translate="no"><a href="/reference/android/telephony/ims/RegistrationManager.RegistrationCallback">RegistrationManager.RegistrationCallback</a></code>.

 When the subscription associated with this callback is removed (SIM removed, ESIM swap,
 etc...), this callback will automatically be removed. If this method is called for an
 inactive subscription, it will result in a no-op.

 Requires Permission: <code dir="ltr" translate="no"><a href="/reference/android/Manifest.permission#READ_PRECISE_PHONE_STATE">READ_PRECISE_PHONE_STATE</a></code> or that the calling app has carrier privileges
 (see <code dir="ltr" translate="no"><a href="/reference/android/telephony/TelephonyManager#hasCarrierPrivileges()">TelephonyManager.hasCarrierPrivileges()</a></code>).

 <br/>
 Requires <code dir="ltr" translate="no"><a href="/reference/android/Manifest.permission#READ_PRECISE_PHONE_STATE">Manifest.permission.READ_PRECISE_PHONE_STATE</a></code></p>
