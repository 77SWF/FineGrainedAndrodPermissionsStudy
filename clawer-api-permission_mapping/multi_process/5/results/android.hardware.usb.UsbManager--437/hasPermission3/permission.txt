<p>Returns true if the caller has permission to access the device.
 Permission might have been granted temporarily via
 <code dir="ltr" translate="no"><a href="/reference/android/hardware/usb/UsbManager#requestPermission(android.hardware.usb.UsbDevice,%20android.app.PendingIntent)">requestPermission(android.hardware.usb.UsbDevice, android.app.PendingIntent)</a></code> or
 by the user choosing the caller as the default application for the device.
 Permission for USB devices of class <code dir="ltr" translate="no"><a href="/reference/android/hardware/usb/UsbConstants#USB_CLASS_VIDEO">UsbConstants#USB_CLASS_VIDEO</a></code> for clients that
 target SDK <code dir="ltr" translate="no"><a href="/reference/android/os/Build.VERSION_CODES#P">Build.VERSION_CODES.P</a></code> and above can be granted only if they
 have additionally the <code dir="ltr" translate="no"><a href="/reference/android/Manifest.permission#CAMERA">Manifest.permission.CAMERA</a></code> permission.

 <br/>
