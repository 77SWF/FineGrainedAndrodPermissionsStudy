<p>Returns true if the caller has permission to access the accessory.
 Permission might have been granted temporarily via
 <code dir="ltr" translate="no"><a href="/reference/android/hardware/usb/UsbManager#requestPermission(android.hardware.usb.UsbAccessory,%20android.app.PendingIntent)">requestPermission(android.hardware.usb.UsbAccessory, android.app.PendingIntent)</a></code> or
 by the user choosing the caller as the default application for the accessory.
