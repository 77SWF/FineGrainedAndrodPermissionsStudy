<p>Possible return values are
 <code dir="ltr" translate="no"><a href="/reference/android/bluetooth/BluetoothAdapter#STATE_OFF">STATE_OFF</a></code>,
 <code dir="ltr" translate="no"><a href="/reference/android/bluetooth/BluetoothAdapter#STATE_TURNING_ON">STATE_TURNING_ON</a></code>,
 <code dir="ltr" translate="no"><a href="/reference/android/bluetooth/BluetoothAdapter#STATE_ON">STATE_ON</a></code>,
 <code dir="ltr" translate="no"><a href="/reference/android/bluetooth/BluetoothAdapter#STATE_TURNING_OFF">STATE_TURNING_OFF</a></code>.

 <br/>
 For apps targeting <code dir="ltr" translate="no"><a href="/reference/android/os/Build.VERSION_CODES#R">Build.VERSION_CODES#R</a></code> or lower, this
 requires the <code dir="ltr" translate="no"><a href="/reference/android/Manifest.permission#BLUETOOTH">Manifest.permission#BLUETOOTH</a></code> permission
 which can be gained with a simple <code dir="ltr" translate="no">&lt;uses-permission&gt;</code>
 manifest tag.</p>
