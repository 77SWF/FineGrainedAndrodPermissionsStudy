<p>After all characteristics have been queued up and verified,
 <code dir="ltr" translate="no"><a href="/reference/android/bluetooth/BluetoothGatt#executeReliableWrite()">executeReliableWrite()</a></code> will execute all writes. If a characteristic
 was not written correctly, calling <code dir="ltr" translate="no"><a href="/reference/android/bluetooth/BluetoothGatt#abortReliableWrite()">abortReliableWrite()</a></code> will
 cancel the current transaction without committing any values on the
 remote device.

 <br/>
 For apps targeting <code dir="ltr" translate="no"><a href="/reference/android/os/Build.VERSION_CODES#R">Build.VERSION_CODES#R</a></code> or lower, this

 which can be gained with a simple <code dir="ltr" translate="no">&lt;uses-permission&gt;</code>
 manifest tag.

 permission which can be gained with
