<pre class="api-signature no-pretty-print" dir="ltr" translate="no">public abstract boolean checkPermission (<a href="/reference/java/security/Principal">Principal</a> principal, 
                <a href="/reference/java/security/acl/Permission">Permission</a> permission)</pre><p>Checks whether or not the specified principal has the specified
 permission. If it does, true is returned, otherwise false is returned.

 More specifically, this method checks whether the passed permission
 is a member of the allowed permission set of the specified principal.
 The allowed permission set is determined by the same algorithm as is
 used by the <code dir="ltr" translate="no">getPermissions</code> method.</p><table class="responsive">
