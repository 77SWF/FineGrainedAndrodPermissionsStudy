<p>Removes an existing <code dir="ltr" translate="no"><a href="/reference/android/telephony/ims/RegistrationManager.RegistrationCallback">RegistrationCallback</a></code>.

 When the subscription associated with this callback is removed (SIM removed, ESIM swap,
 etc...), this callback will automatically be removed. If this method is called for an
 inactive subscription, it will result in a no-op.

 <br/>
 Requires android.Manifest.permission.READ_PRIVILEGED_PHONE_STATE</p>
